# Introduction to Git and GitHub

## Simple Interest Calculator

This project contains a Bash script that calculates simple interest based on user input.

The user provides:

- Principal amount
- Annual rate of interest
- Time period in years

The script then calculates the simple interest using the formula:

Simple Interest = Principal × Rate × Time / 100

## Project Purpose

This project is part of the final assignment for the Introduction to Git and GitHub course. It demonstrates working with GitHub UI, Git CLI, branches, commits, merges, pull requests, and repository documentation files.
